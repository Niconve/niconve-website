Akun & Login

Bagaimana cara membuat akun di Niconve?

Untuk membuat akun di Niconve:

  • Klik tombol "Daftar" di pojok kanan atas
  • Isi formulir pendaftaran dengan nama, email, dan password
  • Klik "Daftar" dan Anda akan otomatis login
  • Selamat! Akun Anda sudah aktif dan siap digunakan
Saya lupa password, bagaimana cara reset?

Untuk reset password:

  • Klik "Lupa Password" di halaman login
  • Masukkan email Anda yang terdaftar
  • Cek email Anda untuk link reset password
  • Klik link tersebut dan buat password baru

Catatan: Link reset password berlaku selama 1 jam.

Apakah akun Niconve gratis?

Ya! Pendaftaran akun di Niconve 100% gratis. Anda hanya perlu membayar untuk aplikasi berbayar yang ingin Anda beli. Aplikasi gratis dapat diunduh tanpa biaya apapun.

Pembayaran

Metode pembayaran apa saja yang tersedia?

Kami menyediakan berbagai metode pembayaran melalui Midtrans:

  • Kartu Kredit/Debit: Visa, Mastercard, JCB
  • Transfer Bank: BCA, Mandiri, BNI, BRI, Permata
  • E-Wallet: GoPay, OVO, ShopeePay, DANA
  • Retail: Alfamart, Indomaret
Apakah pembayaran di Niconve aman?

Ya, sangat aman! Kami menggunakan Midtrans sebagai payment gateway yang memiliki sertifikasi PCI DSS Level 1 (standar keamanan tertinggi untuk payment gateway). Data kartu kredit Anda tidak pernah disimpan di server kami.

Berapa lama proses verifikasi pembayaran?

Waktu verifikasi pembayaran tergantung metode yang digunakan:

  • Kartu Kredit/E-Wallet: Instant (real-time)
  • Transfer Bank: 5-10 menit
  • Retail Store: 5-10 menit setelah pembayaran di toko

Setelah pembayaran terverifikasi, Anda akan mendapat email konfirmasi dan aplikasi langsung bisa diunduh.

Apakah ada biaya tambahan?

Tidak ada biaya tersembunyi! Harga yang tertera adalah harga final yang harus Anda bayar. Namun, beberapa bank mungkin mengenakan biaya admin untuk transfer bank (biasanya Rp 1.000-5.000).

Apakah bisa refund jika tidak puas?

Kami menyediakan garansi uang kembali 7 hari jika Anda tidak puas dengan aplikasi yang dibeli, dengan syarat:

  • Aplikasi tidak berfungsi dengan baik
  • Fitur tidak sesuai deskripsi
  • Bug/error yang tidak bisa diperbaiki

Untuk mengajukan refund, hubungi customer support dengan menyertakan order ID dan alasan refund.

Aplikasi

Bagaimana cara download aplikasi?

Untuk aplikasi gratis:

  • Browse aplikasi di halaman "Aplikasi"
  • Klik tombol "Download Gratis"
  • File APK akan terunduh otomatis

Untuk aplikasi berbayar:

  • Login ke akun Anda
  • Klik "Beli Sekarang" pada aplikasi yang ingin dibeli
  • Pilih metode pembayaran dan selesaikan pembayaran
  • Setelah verifikasi, aplikasi akan muncul di Dashboard Anda
  • Klik "Download" untuk mengunduh aplikasi
Apakah aplikasi yang dibeli bisa di-download berkali-kali?

Ya! Setelah Anda membeli aplikasi, Anda mendapat lifetime access. Artinya:

  • Download unlimited (tidak dibatasi)
  • Berlaku selamanya
  • Bisa download di device berbeda
  • Gratis update aplikasi

Anda bisa login ke Dashboard kapan saja untuk re-download aplikasi yang sudah Anda beli.

Apakah aplikasi aman dari virus/malware?

100% aman! Semua aplikasi yang diupload di Niconve:

  • Melalui proses verifikasi admin
  • Di-scan dengan antivirus
  • Bebas malware dan virus
  • Dari developer terpercaya

Kami menjamin keamanan semua aplikasi di platform kami.

Apa itu aplikasi Beta?

Aplikasi Beta adalah versi pre-release yang masih dalam tahap testing. Karakteristiknya:

  • Gratis untuk periode testing (meskipun aplikasi final berbayar)
  • Mungkin ada bug atau fitur yang belum sempurna
  • Feedback Anda sangat membantu developer
  • Setelah periode beta berakhir, aplikasi akan menjadi versi Full Release

Jika Anda download aplikasi saat beta, Anda akan otomatis mendapat update ke versi final gratis!

Bagaimana cara install aplikasi APK?

Untuk install aplikasi APK di Android:

  • Download file APK dari Dashboard Anda
  • Buka file manager di HP Anda
  • Cari file APK yang baru didownload (biasanya di folder "Download")
  • Tap file APK untuk install
  • Jika muncul peringatan "Unknown Sources", aktifkan izin install dari sumber tidak dikenal
  • Ikuti instruksi instalasi
  • Selesai! Aplikasi siap digunakan
